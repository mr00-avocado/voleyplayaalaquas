# Voley Playa Alaquàs - WordPress

Esta carpeta contiene un tema WordPress listo para usar en un hosting real con dominio propio.

## Qué incluirá la web
- Portada con llamada a la acción
- Sección de servicios
- Formulario de inscripción
- Contacto con teléfono y correo

## Cómo publicarla con dominio
1. Compra o reserva un dominio (por ejemplo, voleyplayaalaquas.com).
2. Elige un hosting WordPress (Hostinger, SiteGround, WordPress.com, Cloudways, etc.).
3. Instala WordPress en el hosting.
4. Sube el archivo ZIP del tema desde Apariencia > Temas > Añadir nuevo.
5. Activa el tema.
6. Configura la página principal y el menú.
7. Enlaza el dominio en la zona DNS del proveedor.

## Nota importante
El formulario envía la solicitud por correo desde WordPress. En muchos hostings puede requerir un plugin de correo como WP Mail SMTP para que funcione correctamente.
